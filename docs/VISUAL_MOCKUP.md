## 🎬 Animation Sequence

### Page Load
```
Time    Event
────────────────────────────────────
0ms     → Background starts rendering
100ms   → Particles fade in
200ms   → Logo scales in (0.8 → 1.0)
300ms   → Logo glow pulse begins
400ms   → Headline slides up + fades in
600ms   → Subtitle slides up + fades in
800ms   → CTA button slides up + glows
```

### Scroll Interactions
```
Section          Trigger            Animation
──────────────────────────────────────────────────────
Problem          50% visible     → Cards stagger in (left to right)
Solution         50% visible     → Cards stagger in (delay 150ms each)
Impact           50% visible     → Cards fade in + counters start
Waitlist         50% visible     → Form slides up, fields stagger
Footer           Any visible     → Fade in
```

### Hover States
```
Element          Hover Effect
───────────────────────────────────
GlowButton    → Glow intensifies, slight scale (1.05)
GlassCard     → Border glow appears, backdrop blur increases
StatCard      → Scale (1.02), glow pulse
FeatureCard   → Rotate slightly (3deg), glow border
Logo          → Glow pulse speed increases
```

## ✨ Glow Effects Guide

### Subtle Glow (Always On)
```css
box-shadow: 0 0 20px rgba(0, 212, 255, 0.3);
```
Used for: Logo, section borders

### Medium Glow (Hover)
```css
box-shadow: 0 0 30px rgba(0, 212, 255, 0.5);
```
Used for: Buttons, cards on hover

### Strong Glow (Active/Focus)
```css
box-shadow: 
  0 0 40px rgba(0, 212, 255, 0.6),
  0 0 80px rgba(0, 212, 255, 0.3);
```
Used for: CTA button, active form field
